Versión Laravel 8.

Utilizar Xampp con php 7.4 o una versión mayor.

Base de Datos MySQL.

Copiar carpeta de proyecto a carpeta htdocs de xampp, crear base de datos productos en phpMyAdmin e importar base de datos.

En la carpeta database del proyecto se encuentra el archivo productos.sql

La base de datos no cuenta con contraseña.

En la barra de URL del navegador colocar http://localhost/productos/public

Usuario: admin@gmail.com

En el menú del sistema están las opciones de login y registro de usuarios.

Tabla de registros de productos utiliza livewire para consulta de datos.

Sistema de registro y login de usuarios se utiliza jetstream.

Plantilla de administrador se utiliza AdminLTE.

Select multiple se utiliza Select2.

Plugin de textarea se utiliza ckeditor.

Se utiliza plantilla phanton de HTML5up.net

Se utiliza relación polimorfica para almacenar rutas de imagenes.
